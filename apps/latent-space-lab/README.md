# Latent Space Lab (Application 1)

This folder marks the Maxey0 platform application boundary for the Latent Space Lab.

Current implementation lives in the root Next.js workspace (`src/`, `data/`, `docs/`, `scripts/`) for compatibility with existing scripts and CI.

As more applications are added, each app should receive its own directory under `apps/` while sharing common platform datasets, docs, and research artifacts.
