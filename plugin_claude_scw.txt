========================================================================================
RUNTIME ORDER (MINIMAL MILESTONE: CLAUDE DESKTOP HOST + MAXEY0 MCP + 2 DOWNSTREAM PLANES)
========================================================================================

(1) Claude Desktop starts -> user enables Maxey0 MCP server connection
(2) Claude MCP client discovers Maxey0: tools/resources/prompts
(3) User triggers Maxey0: scw.create(SCW0, determinism=strict|bounded, tick_logging=on)
(4) Maxey0 connects (as MCP client) to downstream plane A + plane B (two MCP servers)
(5) Maxey0 builds capability graph + policy envelopes for each plane
(6) SCW0 runs tick loop; requests tool access -> Maxey0 issues (GATE#) per plane
(7) Dispatcher calls downstream tools via (GATE#); results return with digests
(8) SCW0 seals run: CST + ledger entry + replay recipe
(9) Claude Desktop displays result + optional: opens Maxey0 dashboard panels (trace/gates)
